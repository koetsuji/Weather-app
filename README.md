# Search Weather

A small weather app built for fun.

The first version was made in 2023 with the help of ChatGPT 3.5.
The UI was rough, the code was chaotic, but it worked.

In 2026, the project was rebuilt and modernized with ChatGPT 5.5:
- redesigned mobile-friendly UI
- animated SVG weather icons
- dynamic backgrounds based on weather conditions
- city autocomplete
- geolocation support
- cleaner frontend structure

Maybe in 2029 this thing will be rebuilt again with whatever absurd LLMs exist by then ;)
